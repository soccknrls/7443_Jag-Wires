# 7443_Jag-Wires
Benchmark for SDS MK3 Modules using (2) Neos and SRX Mag Encoders

## CAN Assignment
###### Module 1 (Front Right)
* Driving Motor = 1
* Turning Motor = 2
###### Module 2 (Back Right)
* Driving Motor = 3
* Turning Motor = 4
###### Module 3 (Back Left)
* Driving Motor = 5
* Turning Motor = 6
###### Module 4 (Front Left)
* Driving Motor = 7
* Turning Motor = 8

* Channel A = 1
* Channel B = 2
###### Module 2 (Back Right)
* Channel A = 3
* Channel B = 4
###### Module 3 (Back Left)
* Channel A = 5
* Channel B = 6
###### Module 4 (Front Left)
* Channel A = 7
* Channel B = 8
